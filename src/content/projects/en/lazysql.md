Lazygit-inspired terminal UI for SQL databases: browse connections, schemas and tables, and run queries through a vim-mode editor with keyboard-driven panel navigation. A unified Go driver layer covers PostgreSQL, MySQL and SQLite, with SSH tunnels, per-connection saved queries and mandatory confirmation on destructive statements.
